SpendSave is a full-stack expense tracking web app.
Track income and expenses by categories with real-time balance updates.
Set budgets and savings goals to manage spending.
View analytics and stats to analyze your transactions over time.

Features:
User Authentication:Secure sign up and login system.
Income & Expense Tracking:Users can add income and expenses with categories and notes.
Real-Time Balance Calculation:Automatically calculates total income, expenses, and current balance.
Insufficient Balance Protection:Prevents users from spending more money than they have.
Edit & Delete Transactions :Users can update or remove transactions anytime.
Analytics Dashboard:Visual charts showing spending and income breakdowns.
Responsive UI:Works smoothly on mobile and desktop devices.

Tech Stack
Frontend: React, Tailwind CSS, Recharts
Backend: Node.js, Express
Database: MongoDB Atlas
Authentication: JWT (JSON Web Tokens)

How It Works (Simple)
Users create an account or log in
Add income and expense records
App calculates balance automatically
Expenses cannot exceed available balance
Charts help visualize spending habits
